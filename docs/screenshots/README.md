# Screenshot Guide

Save polished README screenshots in this folder using these filenames:

- `overview-desktop.png`
- `geography-tab.png`
- `equity-lens.png`
- `mobile-view.png`

Suggested source:

- Live app: `https://vaccination-coverage-dashboard.onrender.com/`

Capture tips:

- Use the deployed app, not localhost.
- Prefer PNG for clarity in GitHub README files.
- Keep width consistent across desktop shots.
- Crop to the app content unless the browser frame adds value.
